## Full Stack React Web app with large functionality and responsive
### Important Features:
- Backend Apis with express and node js
- Jsonwebtoken library
	- JWT token using cookies
- bcryptjs library
- Styled Component
  - Dark & Light mode
-  Firebase:
	- Storage
	- Auth
-  Redux Toolkit
	-  Store
	-  Slice
-  Redux Persist
-  Material UI
-  timeago.js library
- Proxy instead of CORS
